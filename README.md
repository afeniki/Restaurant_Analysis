# Restaurant Business Analytics Dashboard (Power BI)

## Project Overview
This project analyzes restaurant data to explore patterns in restaurant distribution, pricing, customer characteristics, and ratings. The analysis is presented through interactive Power BI dashboards to summarize key metrics and support exploratory business analysis.

## Analysis Focus
The dashboards examine:

- Distribution of restaurants across locations and cuisines
- Restaurant pricing ranges and associated characteristics
- Customer distribution and preferences
- Restaurant ratings (food, service, and overall)
- Franchise and non-franchise restaurant breakdown
- Availability of service features such as parking, alcohol, and smoking areas

## Key Metrics
The analysis includes summary metrics such as:

- Total number of restaurants
- Number of cuisines
- Number of states and cities represented
- Customer count
- Restaurant ratings
- Price range distribution
- Franchise vs non-franchise counts

## Dashboard Features
- Interactive filters for cuisine, city, state, price range, and service attributes
- KPI summary cards displaying high-level counts
- Visualizations showing distribution by category, location, and pricing
- Rating comparisons across different restaurant segments

## Tools Used
- Power BI
